# Comando: /tarefa

Trabalhe na próxima tarefa disponível.

## O que fazer:

1. Execute `pnpm task next` para iniciar a próxima tarefa

2. Leia os detalhes da tarefa

3. Planeje a implementação:
   - Quais arquivos serão modificados?
   - Quais dependências são necessárias?
   - Há riscos ou considerações especiais?

4. Implemente a tarefa passo a passo

5. Após concluir:
   - Execute `pnpm lint` para verificar
   - Execute `pnpm typecheck` para verificar tipos
   - Execute `pnpm task done <id>` para marcar como concluída

6. Sugira a mensagem de commit

## Regras:
- Sempre teste o código antes de marcar como concluído
- Explique o que está fazendo em cada passo
- Responda em português brasileiro
