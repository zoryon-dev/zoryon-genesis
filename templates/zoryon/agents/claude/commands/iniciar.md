# Comando: /iniciar

Inicie o desenvolvimento do projeto Zoryon Genesis.

## O que fazer:

1. **Leia as instruções:** `.zoryon/flow/PARA-IA-LER.md`
   - Este arquivo contém o contexto completo do projeto

2. **Verifique se existe briefing:** `.zoryon/flow/BRIEFING.md`
   - Se existir: leia para entender a ideia do usuário
   - Se não existir: pergunte o que ele quer construir

3. **Se tiver briefing, ofereça opções:**
   ```
   Entendi! Você quer criar [resumo]. Posso te ajudar de algumas formas:

   1. Expandir em um PRD completo
   2. Criar user stories detalhadas
   3. Criar tarefas práticas para começar
   4. Ir direto pro código

   O que prefere?
   ```

4. **Se NÃO tiver briefing:**
   ```
   Olá! Vi que seu projeto foi criado com Zoryon Genesis.

   Você ainda não definiu o que vai construir.
   Me conta: O que você quer criar?
   ```

5. **Execute o que o usuário escolher**
