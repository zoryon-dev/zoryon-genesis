# Comece Aqui

Bem-vindo ao seu projeto Zoryon Genesis!

---

## 3 Passos para Começar

### 1. Instalar

```bash
pnpm install
```

> Não tem pnpm? `npm install -g pnpm`

### 2. Configurar

```bash
cp .env.example .env
```

Edite `.env` com suas credenciais (se aplicável).

### 3. Iniciar

```bash
pnpm dev
```

Acesse [http://localhost:3000](http://localhost:3000)

---

## Usando com IA

Abra o projeto com Claude Code, Cursor ou sua IA e diga:

> "Leia `.zoryon/flow/PARA-IA-LER.md` e me ajude a começar"

A IA vai:
1. Entender seu projeto
2. Ver se você tem um briefing
3. Te guiar no desenvolvimento

---

## Comandos Básicos

| Comando | O que faz |
|---------|-----------|
| `pnpm dev` | Iniciar servidor |
| `pnpm task list` | Ver tarefas |
| `pnpm task next` | Próxima tarefa |
| `pnpm task done 1` | Concluir tarefa |
